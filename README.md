# QUICKSTART

## Installation & Setup


**1.** **`Clone`** this repo
```sh
git clone https://github.com/geloman-likes-rust/neovim.git ~/.nvim
```

**2.** Run **`setup`** script
```sh
~/.nvim/setup.sh
```

**3.** **`Refresh`** shell session
```sh
source ~/.bashrc
# or source ~/.zshrc if your default shell is zsh
```

**4.** Sanity **`check`**
```
nvim
```
