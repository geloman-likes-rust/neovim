# QUICKSTART

## Installation & Setup


**1.** **`Clone`** this repo
```
git clone https://github.com/geloman-likes-rust/neovim.git ~/.nvim
```

**2.** Run **`setup`** script
```
~/.nvim/setup.sh
```

**3.** **`Refresh`** shell session
```
source ~/.bashrc
```

**4.** Sanity **`check`**
```
nvim
```
